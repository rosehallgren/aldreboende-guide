Ansökningsprocessen

Så får Rolf en plats på vård- och omsorgsboende i Stockholm:

  1. Ring Äldre direkt: 08-80 65 65Be att få prata med en biståndshandläggare i Rolfs stadsdel (Spånga-Tensta). Rose kan ringa som anhörig.
  2. Personligt möte hemma hos RolfBiståndshandläggaren bokar möte. Rose bör vara med. Beskriv Rolfs vardag: vaskulär demens, amputerat ben, rullstolsburen, fallrisk, upprepade sjukhusvistelser. Ta med läkarintyg och medicinlista.
  3. Utredning → biståndsbeslut (2–4 veckor)Handläggaren utreder behoven. Med Rolfs situation (dygnet runt-behov, kan inte bo kvar) bör beslutet komma relativt snabbt.
  4. Välj boendeStockholm har valfrihetssystem — ni har rätt att välja! Ange flera alternativ. Använd denna guide.
  5. Erbjudande och inflyttningNi kan tacka nej till ett erbjudande utan att förlora köplats. Vid inflyttning upprättas en genomförandeplan (vårdplan).

Vid avslag: Överklaga skriftligt till socialnämnden inom 3 veckor. Gratis. Se rättigheter nedan.

Checklista vid besök

Ta med vid studiebesök. Fokusera på det viktigaste för Rolf:

Rolf-specifikt (fråga alltid)

  • Erfarenhet av rullstolsburna med amputation?
  • Sjukgymnast — hur ofta? Rehabiliteringsprogram?
  • Arbetsterapeut för hjälpmedelsanpassning?
  • Hur hanteras förflyttning säng ↔ rullstol?
  • Demensutbildning hos personalen? Certifieringar?
  • Erfarenhet av vaskulär demens / stroke-uppföljning?

Allmänt

  • Första intryck — atmosfär, lukt, bemötande?
  • Lägenheten — storlek, badrum, rullstolsanpassat?
  • Personal — tid för boende? Bemanning kvällar/nätter?
  • Mat — be om menyer, delta vid måltid om möjligt
  • Aktiviteter — be om program senaste veckorna (inte broschyren)
  • Läkarbesök — hur ofta? Dygnet-runt-beredskap?
  • Besökstider — fria? Kan anhöriga äta med?

Rättigheter

  • Rätt till individuell bedömning
  • Rätt att välja boende (valfrihetssystemet)
  • Rätt till genomförandeplan (vårdplan)
  • Rätt till värdigt bemötande och självbestämmande

Vid avslag

Skriv överklagande till socialnämnden inom 3 veckor. De skickar vidare till förvaltningsrätten. Helt gratis.
